# PitchPerfect

PitchPerfect is a project for the **Udacity iOS Developer Nanodegree**.

Users can record voices using the device microphone and are then able to play the recorded voice with differenct modulations:
- **Slow:** 0.5 rate
- **Fast:** 1.5 rate
- **Heigh:** 1000 pitch
- **Low:** 0.5 -1000 pitch
- **Echo**
- **Reverb**

## Prerequisites

- Xcode: 9.1
- Swift: 4.0
